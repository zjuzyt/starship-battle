# Starship Battle / 星际战舰小游戏

一个零依赖的浏览器 Canvas 小游戏。直接打开 `index.html` 即可游玩。

## 在线试玩

启用 GitHub Pages 后，这里可以填写在线地址：

```text
https://<你的 GitHub 用户名>.github.io/starship-battle/
```

## 操作

- `WASD` 或方向键：移动战舰
- `空格`：发射
- `P`：暂停或继续
- 触屏设备可使用页面底部的方向键和发射按钮

## 目标

击毁敌舰和陨石获得分数，拾取蓝色补给可以短时间三连发。生命归零后可以重新开始。

## 本地运行

直接打开 `index.html`，或启动一个本地静态服务器：

```bash
python3 -m http.server 8000
```

然后访问：

```text
http://127.0.0.1:8000
```
